# linear-solver

## Prerequisites

- Docker
- docker-compose

## Installation

1. Download HSL source from [here](https://www.hsl.rl.ac.uk/ipopt/) (you should select `Source` rather than `Linux x86_64`)
2. Put the `coinhsl-20xx.xx.xx.tar.gz` into `data` directory.
3. Run `docker-compose up --build`
